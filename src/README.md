# Source Code

This directory contains the core logic for the NBA Data Hub.

## Directory Structure

- **[`cli/`](cli/README.md)**: Command-line interface scripts to run the pipeline.
- **[`core/`](core/README.md)**: Core infrastructure, configuration, and database connection management.
- **[`etl/`](etl/README.md)**: Extract, Transform, Load pipelines for data ingestion and processing.
- **[`frontend/`](frontend/README.md)**: Streamlit-based web interface (if applicable) or query layers.
- **[`scraping/`](scraping/README.md)**: Web scrapers for fetching historical data from external sources.
- **[`utils/`](utils/README.md)**: Shared utility functions and helpers.
